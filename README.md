# Simple Node Server

A minimal Node.js HTTP server example used for a CI/CD pipeline workshop.

## Getting Started

**Install dependencies**
```bash
npm install
```

**Run the server**
```bash
npm start
```

Open your browser at [http://localhost:3000](http://localhost:3000) — you should see some text displayed!

## Other Commands

| Command | Description |
|---|---|
| `npm test` | Run Jest unit tests |
| `npm run lint` | Run ESLint |
| `npm run build` | Empty placeholder command for building the application |
